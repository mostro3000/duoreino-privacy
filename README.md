# duoreino-privacy

Política de privacidad de la app **DuoReino** (`ar.com.mt0.duoreino`), publicada en Google Play.

📄 Página pública: **https://mostro3000.github.io/duoreino-privacy/**

DuoReino no pide registro ni recopila datos personales. Desde la versión 1.2 usa
**Firebase Analytics** para estadísticas de uso **anónimas** (eventos de partida,
modelo/versión del dispositivo, país aproximado), por lo que requiere conexión a
internet. No muestra anuncios ni comparte datos con fines publicitarios.
